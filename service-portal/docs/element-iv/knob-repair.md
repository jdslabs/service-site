# Knob Repair

Step-by-step guide to remove, replace, or repair the Element IV volume knob.

## Tools Required

- 1.5mm hex key (Allen wrench)
- Soft cloth or microfiber towel

## Difficulty

**Moderate** — This procedure requires careful handling to avoid scratching the enclosure.

## Procedure

### Step 1: Power Off

Disconnect power and all cables from Element IV.

### Step 2: Locate the Set Screw

The knob is secured by a small set screw on the underside.

1. Gently tilt Element IV to view the bottom of the knob
2. Locate the 1.5mm hex set screw

### Step 3: Loosen the Set Screw

1. Insert the 1.5mm hex key into the set screw
2. Turn counter-clockwise 2-3 full rotations
3. Do not fully remove the screw — just loosen it

:::warning
Do not over-tighten or strip the set screw. Light pressure is sufficient.
:::

### Step 4: Remove the Knob

1. Gently pull the knob straight off the shaft
2. If stuck, wiggle slightly while pulling
3. Place the knob on a soft surface to prevent scratches

### Step 5: Install Replacement Knob

1. Align the knob with the shaft
2. Push firmly until seated
3. Rotate to find the set screw detent on the shaft
4. Tighten the set screw until snug (do not over-tighten)

### Step 6: Test

1. Reconnect power
2. Verify smooth rotation
3. Test press function (output toggle)
4. Test long-press function (menu access)
