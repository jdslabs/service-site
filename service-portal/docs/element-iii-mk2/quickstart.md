# Quickstart

Get your Element III Mk2 up and running in just a few minutes.

## What's in the Box

- Element III Mk2 DAC/Amp
- USB cable
- Power adapter
- Quick start card

## Setup

### 1. Connect Power

Plug the power adapter into Element III Mk2 and connect to a wall outlet.

### 2. Connect to Your Computer

Use the included USB cable to connect Element III Mk2 to your computer.

### 3. Connect Headphones

Plug your headphones into the front 6.35mm (1/4") or 3.5mm jack.

### 4. Set as Default Audio Device

**Windows:** Right-click the speaker icon → Sound settings → Select "Element III Mk2" as output device.

**Mac:** System Preferences → Sound → Output → Select "Element III Mk2".

### 5. Adjust Volume

Turn the knob to adjust volume. Press to switch between headphone and line output.

## Controls

### Volume Knob

- **Rotate:** Adjust volume
- **Press:** Toggle between headphone and line output
- **Long Press:** Access menu

### Menu Navigation

1. Long press the knob to enter the menu
2. Rotate to navigate options
3. Press to select
4. Long press to exit
