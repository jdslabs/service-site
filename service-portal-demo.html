X200 Motor Replacement

Moderate 45 – 60 min Part No. MOT-X2-440 In Stock
⚠ Safety Warning Disconnect and lock out the power supply before beginning. Capacitors may hold charge for up to 3 minutes after power-off.

X200 Motor Replacement — Full Walkthrough (8:42)

Embed your YouTube or Vimeo URL here. No hosting required.

🔧 Tools Required
  • T25 Torx screwdriver
  • 5mm hex key
  • Needle-nose pliers
  • Multimeter
  • Torque wrench (5–8 Nm)
📦 Parts & Materials
  • Motor assembly MOT-X2-440
  • Gasket set GSK-002 (1×)
  • Threadlocker (medium)
  • Dielectric grease
Procedure
1
Remove the access panel
Using the T25 Torx driver, remove the four M4 screws from the rear access panel. Set the panel aside — the gasket is reusable if undamaged.
[ Photo: Rear panel with screw locations highlighted ]
2
Disconnect motor wiring harness
Locate the 6-pin connector (white, marked M1) on the left side of the motor. Press the release tab and pull straight back. Do not twist.
[ Photo: Connector location with release tab indicated ]
3
Remove the motor mounting bolts
Using the 5mm hex key, remove all three mounting bolts (M6×20mm). Support the motor as you remove the final bolt — it will become free.
💡 Tip Mark the motor orientation with a marker before removal — the replacement must be installed in the same orientation.
4
Install replacement motor
Position the new motor (MOT-X2-440) in the same orientation. Hand-tighten all three bolts before torquing to 6 Nm in a star pattern. Apply a small amount of threadlocker to each bolt.
[ Photo: Torque sequence diagram ]
5
Reconnect and test
Reconnect the wiring harness until it clicks. Replace the access panel and torque the four screws to 3 Nm. Restore power and run a 30-second no-load test before returning to service.

Getting Started with Docusaurus

Everything you need to go from zero to a live service portal at service.jdslabs.com

Install Node.js

Download Node.js (v18+) from nodejs.org. This is the only software prerequisite.

Scaffold your Docusaurus project

Run npx create-docusaurus@latest service-portal classic in your terminal. This generates the full folder structure below.

Replace the config with yours

Edit docusaurus.config.js — set your site name, URL, and subdomain. See the config sample below.

Drop your content into /docs

Each guide is a plain .md (Markdown) file. Paste your existing content in, add a frontmatter block at the top, and embed your video URL.

Deploy to Netlify (free)

Push to GitHub, connect the repo at netlify.com, and point your DNS service subdomain at Netlify. Live in under 30 minutes.
